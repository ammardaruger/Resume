Adaptive video conversion scoring platform ▼

FastAPI service that scores source video quality to auto-crop it into multiple ratio-specific variants for social media, with dynamic cropping, language dubbing, and subtitle generation.

FastAPI · YOLO (Ultralytics) · OpenCV · PaddleOCR · Faster-Whisper · PostgreSQL · JWT · AWS S3 · WebSockets
11+Content sources ingested

Ingests source video (including via MRSS feeds from 11+ content providers) and runs it through a scoring pipeline — ffprobe/frame sampling, OCR text density, face presence, ASR, and YOLO tracking — to compute composite scores that route each clip to reject, conservative, standard, or premium conversion.

  • Dynamic cropping renders each clip into multiple ratio-specific variants for social platforms: YOLO-tracked crop, layout-aware composition for text-heavy frames, or blurred-background "preserve wide" recomposition for portrait/pillarboxed sources.
  • Language conversion pipeline can dub the original audio into a different language and generate subtitles in the target language.
  • JWT-authenticated API with per-user threshold overrides and PostgreSQL-backed conversion history, so async job polling survives multi-worker restarts.
  • Delivered three ways: synchronous HTTP, DB-backed async hooks, and real-time WebSocket status streaming.
  • Deployed as a client freelance engagement, Feb 2026 → present.
